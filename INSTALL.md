# Установка TAT-Secretary

## Для компьютера (Chrome / Edge / Brave)
1. Скачай папку [`extensions/tat-secretary/`](extensions/tat-secretary/)
2. Распакуй архив
3. Открой `chrome://extensions`
4. Включи «Режим разработчика»
5. Нажми «Загрузить распакованное расширение»
6. Выбери папку `tat-secretary`

## Для телефона (Android)
Установи **Kiwi Browser** → `chrome://extensions` → режим разработчика → загрузи папку `tat-secretary`

## Проверка
- Открой `chat.deepseek.com`
- В правом нижнем углу кнопка **🧠 TAT-Secretary**
