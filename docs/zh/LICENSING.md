# TAT 许可概述

| 模块 | 许可证 | 备注 |
|------|--------|------|
| TAT-Defense (核心) | 专有 | 结构异常检测引擎。源代码可供研究使用，商业用途需授权。 |
| TAT-Monitor | AGPL-3.0 | 对话回滚检测。开源，著佐权。 |
| TAT-Diff | AGPL-3.0 | TAT-BRAIN 差分记忆。 |
| TAT-7 (神经网络) | AGPL-3.0 | 可训练的 TAT-7 架构。 |
| Full Defense | 专有 | 五级防御模块。 |
| Fractal Compression | 专有 | 压缩算法。 |
| Memory (Crystal/Holographic) | 专有 | 记忆管理。 |
| Dialog Lifecycle | AGPL-3.0 | 对话生命周期模型。 |
| TAT Modular / Triumvirate / Crystal | 专有 | 多智能体架构。 |
| TAT7-logistics | 专有 | 仓库工业原型。 |

**一般规则：** 核心诊断和预测算法为专有，以保护方法的完整性。支持模块和研究工具在 AGPL-3.0 下开源。
