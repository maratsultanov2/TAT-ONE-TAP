# Лицензирование TAT

| Модуль | Лицензия | Примечания |
|--------|----------|------------|
| TAT-Defense (ядро) | Проприетарная | Движок обнаружения структурных аномалий. Исходный код доступен для исследований, коммерческое использование требует разрешения. |
| TAT-Monitor | AGPL-3.0 | Детекция ревертов в диалогах. Открытый исходный код, копилефт. |
| TAT-Diff | AGPL-3.0 | Дифференциальная память для TAT-BRAIN. |
| TAT-7 (нейросеть) | AGPL-3.0 | Обучаемая архитектура TAT-7. |
| Full Defense | Проприетарная | Пятиуровневый модуль защиты. |
| Fractal Compression | Проприетарная | Алгоритмы сжатия. |
| Memory (Crystal/Holographic) | Проприетарная | Управление памятью. |
| Dialog Lifecycle | AGPL-3.0 | Модель жизненного цикла диалогов. |
| TAT Modular / Triumvirate / Crystal | Проприетарная | Многоагентные архитектуры. |
| TAT7-logistics | Проприетарная | Промышленный прототип для склада. |

**Общее правило:** ключевые алгоритмы диагностики и предсказания являются проприетарными для защиты целостности метода. Вспомогательные модули и исследовательские инструменты открыты под AGPL-3.0.
