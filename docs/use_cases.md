# Use cases for embedded systems

## Wearables
- On‑device classification of motion/heart rate
- Voice command recognition without internet

## Robotics
- Persistent memory across power cycles (`TAT_BRAIN.json`)
- Offline operation, no cloud dependency

## Automotive
- Real‑time driver assistant, low latency (<5ms)
- No network required, fully private

## Home appliances
- User profile storage (preferences, history)
- Runs on cheap microcontrollers

## Why TAT
- Model size <2MB, memory footprint <50KB
- 92.8% accuracy vs 65.5% (logistic regression)
- Zero‑trust by design
